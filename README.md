# CoolKonyha AI Assistant

Ez a projekt egy AI-alapú asszisztens rendszer a CoolKonyha számára, amely segít az ügyfélkommunikációban és az ügymenet követésében.

## Telepítés és Használat

Részletes telepítési útmutatóért lásd a [TELEPITESI_UTMUTATO.md](TELEPITESI_UTMUTATO.md) fájlt.

## Gyorsindítás

1.  Másold a `credentials.json` fájlt a gyökérkönyvtárba.
2.  Töltsd ki a `secrets.toml` fájlt a szükséges adatokkal.
3.  Futtasd a `run_coolkonyha.bat` fájlt.

## Struktúra

*   `app.py`: A fő alkalmazás.
*   `agents/`: AI ügynökök logikája.
*   `utils/`: Segédszkriptek.
*   `ui_config.json`: Felhasználói felület beállításai.
*   `workflow.json`: Üzleti folyamat definíciója.
